Help you manage Nginx local development configuration

Site CLI 是一个帮助你管理和切换 Nginx 本地开发环境配置文件的命令行工具。

Features
--------

+ Site Grouping
+ List Site Information
+ Quick Switching Group/Site/Server
+ Quick Build From Template
